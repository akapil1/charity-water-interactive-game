# 💧 Water Quest Game

An interactive browser-based game inspired by **charity: water**, designed to raise awareness for clean water initiatives through engaging, fast-paced gameplay.

🔗 **Live Demo:** https://akapil1.github.io/water-quest-game/

---

## 📌 Overview

Water Quest Game is a responsive web game where players click charity: water’s signature yellow jerry cans as quickly as possible to earn points before time runs out.

This project combines **interactive JavaScript game logic**, **responsive UI design**, and **cause-driven storytelling** to create a fun and meaningful digital experience.

---

## ✨ Features

- 30-second countdown timer
- Real-time score tracking
- Random jerry can appearance across the grid
- Win / lose game logic based on final score
- Randomized end-game messages
- Replay functionality
- Responsive layout for different screen sizes
- charity: water-inspired branding and styling

## 🎮 How It Works

1. Click the **Start** button to begin the game.
2. Jerry cans appear randomly in the grid.
3. Click each jerry can as quickly as possible to earn points.
4. The timer counts down from 30 seconds.
5. When the timer reaches 0:
   - players with **20 or more points** receive a win message
   - players with **less than 20 points** receive a try-again message
6. Click **Play Again** to restart.
